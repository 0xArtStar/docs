# 积分推荐和返现奖励方案

## 一、 设计思路

**核心痛点** ArtStar 平台致力于打造一个高质量的艺术品交易与投资社区。我们面临的核心挑战是：如何低成本且精准地吸引高净值的藏家、优秀的创作者以及活跃的投资者，同时持续激发用户在平台上的活跃度与交易热情？

**目标用户画像** 我们将用户分为四个主要层级，满足不同人群的需求：

- **S 级（艺术巨擘/机构基石）**：国际知名艺术家、顶级画廊等，为平台提供优质艺术资产和品牌背书。
- **A 级（资深藏家/精英创作者）**：高净值个人、资深买家，是平台稳定的交易主力。
- **B 级（活跃支持者）**：高频参与交易与互动的买家、成长型创作者，为社区注入源源不断的活力。
- **C 级（潜力用户/普通买家）**：小额、低频的购买者，有一定购买力和传播力。
- **D 级（新注册/观望者）**：刚注册尚未交易的新用户，是平台未来的增长潜力。

**整体业务闭环与交互流程**

- **获取专属邀请码**：平台上的任何用户（无论何种身份），都可以一键生成自己的专属邀请链接。
- **分享与注册**：将链接分享给微信好友、社群或社交平台，好友点击链接即可轻松注册，并选择自己的身份（投资者、藏家或创作者）。
- **认证与自动发分**：当好友完成实名认证（KYC）后，系统会立刻自动向邀请人发放“推荐积分”。
- **交易与现金提成**：当好友在平台上购买了艺术品，邀请人不仅能提升自己的等级，还能获得实打实的 USDT 现金返现。
- **升级与特权**：随着积分的不断累积，邀请人的会员等级会不断提升，从而解锁更多平台特权与更高的收益加成，形成“邀请-收益-升级-再邀请”的完美闭环。

**设计原则**

- **与现有架构兼容**：方案必须能无缝融入 Supabase 数据库，并为未来 Web3 身份（钱包地址）绑定做准备。
- **MVP（最小可行产品）导向**：初期以最简单、最可控的方式启动，避免过度开发。
- **人工审核优先**：严格遵循会议要求，在自动化工具成熟前，所有关键决策（尤其是高净值用户评级）由“艺委会”人工完成。
- **一级推荐模型**：仅奖励直接推荐人，坚决杜绝多级分销（传销）风险，简化逻辑。
- **双轨制**：通过标签明确区分 艺术家/收藏家（Creator） 和 投资用户（Collector） 两条轨道。
- **精准获客与价值绑定**：以高激励机制速引 S/A 级等高价值用户入驻；借助多维度积分体系，激励 B/C/D 级参与互动与推广，形成“贡献-回报-忠诚”正向循环。
- **平滑演进**：设计从人工审核到数据驱动，再到链上声誉系统的清晰演进路径。

## 二、 核心架构：用户金字塔模型

基于“资产价值贡献度”（积分/交易额）和“社区影响力”将用户划分为五个层级（以此作为定级标准）：

| 层级      | 用户类型       | 资产/积分门槛   | 描述                 |
| ------- | ---------- | --------- | ------------------ |
| **S 级** | 艺术巨擘/顶级藏家  | 50万+      | 行业领袖，具有极高资产价值和影响力。 |
| **A 级** | 精英创作者/资深藏家 | 20万 - 50万 | 专业人士，持续贡献高质量内容或资金。 |
| **B 级** | 活跃支持者      | 5万 - 20万  | 社区中坚力量，高频参与交易与互动。  |
| **C 级** | 普通买家/潜力创作者 | 1万 - 5万   | 平台活跃用户，有一定购买力和传播力。 |
| **D 级** | 新注册/观望者    | 0 - 1万    | 基础用户，刚注册或轻度参与。     |

## 三、 积分体系与获取机制

平台采用 **“总积分 （ArtPoints， AP）”** 概念，1 AP = 0.05 USD ，**总积分 = 基础任务积分 + 推荐积分 + 交易积分 + 声誉积分**。积分不可转账（Off-chain），存储于 Supabase 数据库，它决定用户的金字塔层级，并在特定活动中可兑换权益或代币空投。

### 1. **基础任务积分**

- **钱包注册账号**：+100 AP（仅限一次）
- **完成 KYC 认证**：+200 AP（仅限一次，需审核通过）

### 2. **推荐积分（拉新奖励）**

邀请好友注册并成功通过实名认证（KYC）后，根据好友的**初始评定等级**发放基础积分。

**基础奖励（按被推荐人层级）：**

- 推荐 **S 级**用户：+10,000 AP （艺委会审核）
- 推荐 **A 级**用户：+5,000 AP（艺委会审核）
- 推荐 **B 级**用户：+2,000 AP （艺委会审核）
- 推荐 **C 级**用户：+500 AP （艺委会审核）
- 推荐 **D 级** 用户：+100 AP（需完成 KYC）

**推荐人加成奖励（按推荐人层级）：** 为了激励下沉市场与活跃用户积极布道，推荐人自身的等级会为推荐积分提供额外加成系数。 _（例如：B 级用户推荐了一个 A 级用户，获得基础 5,000 AP，再加上 B 级的 15% 加成，最终获得 5,750 AP）_

- **D 级**推荐人：+30%（1.3x）
- **C 级**推荐人：+20%（1.2x）
- **B 级**推荐人：+15%（1.15x）
- **A 级**推荐人：+10%（1.1x）
- **S 级**推荐人：无加成

### 3. **交易积分（消费挖矿）**

您自己在平台上申购艺术品，系统会根据交易金额（美元）自动发放积分。等级越高，获取积分的乘数越大，以此鼓励用户升级：

| 用户等级    | 交易积分转化率 (AP / USD) | 积分上限 | 结算周期 |
| ------- | ------------------ | ---- | ---- |
| **D 级** | 1.0 AP / 10 USD    | 无上限  | 随时结算 |
| **C 级** | 1.1 AP / 10 USD    | 无上限  | 随时结算 |
| **B 级** | 1.2 AP / 10 USD    | 无上限  | 随时结算 |
| **A 级** | 1.3 AP / 10 USD    | 无上限  | 随时结算 |
| **S 级** | 1.5 AP / 10 USD    | 无上限  | 随时结算 |

### 4. **声誉积分**

积极参与社区互动、为平台做出特殊贡献的用户，将由“艺委会”人工额外打赏积分。

## 四、 USDT 现金返现（提成）机制

推荐好友不仅能得积分，还能直接获得 USDT 现金提成。平台采用**永久跟单机制**，一次邀请，终身绑定！无论好友是今天买还是明年买，只要他们下单，您都有提成拿。

### 1. **阶梯式返现比例**

根据邀请人**历史累计的推荐交易总额（即所有被邀请人在平台的总消费额）**，实行阶梯式的返现比例，完美对齐用户金字塔模型：

| 累计推荐交易额 (USD) | 对应推荐人业务等级 | 返现比例 | 返现上限 | 结算周期   |
| ------------- | --------- | ---- | ---- | ------ |
| 0 - 10万       | 见习推荐官     | 1.0% | 无上限  | 作品售卖完成 |
| 10万 - 50万     | 初级推荐官     | 1.5% | 无上限  | 作品售卖完成 |
| 50万 - 200万    | 中级推荐官     | 2.0% | 无上限  | 作品售卖完成 |
| 200万 - 500万   | 高级推荐官     | 2.5% | 无上限  | 作品售卖完成 |
| 500万+         | 顶级推荐官     | 3.0% | 无上限  | 作品售卖完成 |

### 2. **KOL / 金牌推荐官专属加成**

如果邀请人是平台签约的推广大使（KOL）或经过“艺委会”认证的“金牌推荐官”，在上述标准提成的基础上，额外享受 **+0.5% ～ +1.0%** 的专属比例加成！ _（例如：中级推荐官原本为 2.0%，若带有 KOL 身份，则可享受 2.5% ～ 3.0% 的返现）_

## 五、 业务流程与风控规则

### 1. **推荐关系绑定**

- **唯一标识**：每个用户生成唯一邀请码 （Invite Code） 和邀请链接。
- **绑定逻辑**：新用户通过链接注册或手动输入邀请码，一旦绑定不可更改。
- **白名单机制**：“艺委会”可授权“金牌推荐官”，其推荐的高净值用户享受快速审核通道。

### 2. **审核与反作弊流程**

- **普通用户 （Collector）**：注册即为 D 级，系统自动发放基础积分。
- **创作者/高净值用户 （Creator）**：
  - 注册后状态为 `Pending`。
  - “艺委会”人工审核背景资料及资产证明。
  - 审核通过后，系统/管理员更新用户层级 （S/A/B/C），并一次性补发对应的高额推荐积分给邀请人。
- **防刷单机制**：USDT 返现需在艺术品交易完全确认且过退款期（如适用）后才可提现结算；若发现恶意刷单或虚假 KYC，平台有权冻结积分与返现账户。
